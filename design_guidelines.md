# Design Guidelines: Personal Page Style

These guidelines promote a simple, clean aesthetic without emojis, using the color palette from the personal portfolio.

## Color Palette

| Usage | Color | Hex |
|-------|-------|-----|
| **Primary** | Blue | `#0066cc` |
| **Primary Hover** | Darker Blue | `#0056b3` |
| **Background** | White | `#ffffff` |
| **Background Sec.** | Light Gray | `#f8f9fa` |
| **Text** | Black | `#000000` |
| **Text Secondary** | Dark Gray | `#333333` |
| **Text Muted** | Medium Gray | `#666666` |
| **Border** | Light Gray | `#e0e0e0` |

## Typography

- **Font Family**: System font stack (Inter/Roboto/Helvetica Neue/Arial).
- **Base Size**: 16px.
- **Headings**: Bold, tight line height. `h1` gradient text allowed.

## Spacing & Layout

- **Grid**: 8px baseline grid.
- **Padding/Margin**: multiples of 4px/8px (`0.5rem`, `1rem`, `1.5rem`).
- **Container**: Max-width ~1200px, responsive padding.

## Components

- **Buttons/Links**: 
  - Primary: Background `#0066cc`, Text White, Radius `4px`.
  - Hover: Background `#0056b3`.
- **Cards**:
  - Background White.
  - Border `#e0e0e0` (1px).
  - Radius `8px`.
  - Shadow: Subtle on hover (`0 4px 6px rgba(0,0,0,0.1)`).

## Rules

1. **No Emojis**: Use SVG icons or text only.
2. **Simple Design**: Avoid excessive decoration, gradients (except main heading), or complex animations.
3. **Consistency**: Adhere strictly to the color palette.
